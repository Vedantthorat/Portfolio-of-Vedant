3. CPU Scheduling Algorithms Simulator ⚡

Jun 2025 | Personal Project

A Python-based simulation of fundamental CPU scheduling algorithms, built using Jupyter Notebook. The project helps students and developers understand how scheduling works in Operating Systems through hands-on implementation.

  • 🧠 Simulates FCFS, Round Robin, Priority Scheduling
  • 📌 User input for arrival time, burst time, priority, and quantum
  • ⏳ Calculates completion, turnaround, and waiting times
  • 📊 Generates Gantt Chart representation
  • 💻 Clean and structured output formatting

Tech Stack: Python (Jupyter Notebook)
